Website ini berdiri sejak 16 Febuari 2022. Website ini bernama Codingku yang diperuntukan untuk para programmer pemula hingga tingkat atas. Website ini menyediakan berbagai kebutuhan bahasa pemrograman secara lengkap dan mudah dipahami.


Codingku sendiri merupakan kursus berbasis web yang dapat diakses secara mudah dan gratis. Website ini dibuat seorang diri oleh Muhammad Feiza Zharfan Priyatnadi dari Universitas Bina Nusantara. Website ini dibuat dengan harapan menambah wawasan para programmer untuk masa depannya.
